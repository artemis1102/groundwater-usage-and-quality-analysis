# Groundwater Usage and Quality Analysis

## Project Overview

This project analyzes regional groundwater usage patterns and groundwater quality across South Korea using public groundwater datasets.

The study aims to identify regional characteristics of groundwater utilization and evaluate groundwater suitability for agricultural use through environmental water quality standards.

## Objectives

- Analyze groundwater usage by purpose (domestic, agricultural, industrial)
- Compare groundwater utilization patterns across regions
- Evaluate groundwater quality using environmental standards
- Identify contamination characteristics by region
- Assess groundwater suitability for agricultural applications

## Dataset

### Groundwater Usage Data (2021–2023)
- Domestic groundwater usage
- Agricultural groundwater usage
- Industrial groundwater usage

### Groundwater Quality Data
- pH
- Nitrate Nitrogen (NO₃-N)
- Chloride
- Cadmium
- Arsenic
- Lead
- Chromium (VI)
- Trichloroethylene (TCE)
- Tetrachloroethylene (PCE)

## Methodology

### 1. Groundwater Usage Analysis
- Data preprocessing using Pandas
- Regional comparison of groundwater consumption
- Trend analysis from 2021 to 2023
- Data visualization using Matplotlib and Seaborn

### 2. Groundwater Quality Assessment
- Water quality standard evaluation
- Contamination detection
- Agricultural water suitability assessment
- Regional contamination pattern analysis

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings

- Groundwater usage patterns vary significantly across regions.
- Agricultural groundwater usage is concentrated in several provinces.
- Groundwater quality differs according to regional environmental characteristics.
- Certain regions exhibit relatively higher contamination risks.
- Water quality indicators can be used to assess agricultural suitability.

## Results

This project provides insights into regional groundwater usage patterns and groundwater quality conditions in South Korea. The findings can support environmental monitoring and agricultural water resource management.
